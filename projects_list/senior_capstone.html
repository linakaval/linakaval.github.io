Senior Capstone - Playlign


Playlign was designed by Justin Haywood, Samantha Shaw, and me for our senior capstone project.

Our project is a web-based application that allows users to request songs to be played in group settings. The problem we are trying to fix is to make playing songs that everyone likes at a party easier and seamless. Users will navigate to the website and be able to request songs for the host to play. These songs will automatically be added to the host's queue, which is connected to Spotify.

As we worked on our project, we followed standards typically followed in the computer science industry. Using agile methodology, we planned sprints, followed the agile workflow, and created a backlog of issues. The technologies we used include Angular, NodeJS, the Spotify API, and Typicode JSON server.
